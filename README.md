# hc-payment-plugin

Stripe and PayPal payment integration for the Apito API platform.

## Features

- Stripe integration
- PayPal support
- Subscriptions
- One-time payments
- Refunds
- Webhook handling

## Installation

```bash
go build -o hc-payment-plugin .
```
